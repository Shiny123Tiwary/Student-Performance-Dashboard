🎓 Student Performance Dashboard | Power BI Project

🚀 Project Highlights

A fully interactive Student Performance Dashboard built in Power BI to analyze academic data and generate meaningful insights for better decision-making.

This project demonstrates skills in:

- 📊 Data Visualization
- 📈 Data Analysis
- 🧹 Data Cleaning (Power Query)
- 🧠 Analytical Thinking

---

📊 Key Features

✔ KPI Cards

- Average Marks
- Total Students
- Pass Percentage

✔ Interactive Visuals

- 📈 Marks Trend Analysis
- 📊 Subject-wise Performance
- 🧑‍🎓 Gender-wise Comparison
- 🏆 Top & Bottom Performers

✔ Dynamic Filters (Slicers)

- Class
- Gender
- Subject
- Date
